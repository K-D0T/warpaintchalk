# War Paint Chalk

**Release Your Inner Warrior** — e-commerce landing site for War Paint Chalk, premium cheer chalk.

- Classic White Chalk — **$10** / 250 ML bottle
- Colored Chalk — **$15** / 250 ML bottle (6 colors)

Orders are placed via an order form that composes an email to **warpaintchalk@yahoo.com** with the order details.

## Tech Stack

- [Next.js](https://nextjs.org) (App Router) + TypeScript
- [Tailwind CSS](https://tailwindcss.com)
- [Framer Motion](https://www.framer.com/motion/) for subtle scroll and hover animations

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploying to Vercel

1. Push this repo to GitHub.
2. Go to [vercel.com/new](https://vercel.com/new) and import the repository.
3. Vercel auto-detects Next.js — no configuration needed. Click **Deploy**.

Every push to the default branch will redeploy automatically.
